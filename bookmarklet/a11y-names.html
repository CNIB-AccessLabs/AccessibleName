Accessible Names Bookmarklet

Self-contained. No network calls. UI is sandboxed in a Shadow DOM so site styles can't bleed in. All UI text is at least 16px for readability.

Install

Drag this button to your bookmarks bar:

a11y: names

Use

  1. Visit the page you want to inspect.
  2. Click the a11y: names bookmark.
  3. Each interactive element gets an outline and a badge above it with its computed accessible name; anything missing a name is red.
  4. Floating panel on the right lists everything. Click a row to scroll and flash that element.
  5. Copy MD puts a Markdown table on your clipboard; DevTools console has the same as console.table plus the raw Markdown.
  6. Click the bookmark again, or hit Close, to clear.
UI isolation. The panel and badges live inside a Shadow DOM with :host { all: initial } and explicit font-family on every selector. Pages that set * { font-family: Lobster } or similar can no longer bleed into the inspector's UI. Outlines on inspected elements are set with !important so site-wide outline: none resets can't suppress them.
Cross-origin iframes. Same-origin policy prevents the bookmarklet from looking inside iframes loaded from a different origin. The panel reports how many frames were skipped. For full coverage across all frames, use the Firefox or Chrome extension.
